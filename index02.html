PROYECTO - PROMEDIO 03

Impresión 3D y prototipo final

27 de Junio, 2023 tls lab

Realizamos la base de nuestro proyecto con la impresora 3D, que sirve para imprimir objetos que tienen volumen y de un material acorde al uso que vayamos a dar, en este caso se utilizó filamento de dos colores. Con una impresora 3D podemos fabricar objetos que nos pueden servir para un uso cotidiano o incluso para reemplazar o mejorar otros.

En este proyecto se realizaron 3 piezas (caja, tapa y nombre), finalmente obtuvimos un gran resultado con el circuito, la base del humificador y el prototipo en 3D.
